# **App Name**: FlashLearn

## Core Features:

- Topic Input: Accept user input of a tech-related topic via a search bar.
- AI Flashcard Generation: Generate a concise flashcard containing a summary, key concepts, real-world example, and learning tip using the Gemini API tool.
- Flashcard Display: Display the AI-generated flashcard in a clean, readable format below the search bar.
- Real-time Information Retrieval: Use Gemini’s web access to retrieve up-to-date information for the generated flashcard.
- Light/Dark Mode Toggle: Allow users to switch between light and dark color themes.

## Style Guidelines:

- Primary color: Deep violet (#673AB7), a modern, intelligent hue drawn from the topic of AI and education, for headers, buttons, and key elements.
- Background color: Light purple (#F3E5F5), a desaturated tint of the primary color that ensures readability while maintaining a cohesive look.
- Accent color: Pink (#E91E63) for interactive elements, like the search bar or mode toggle, lending a bold contrast and implying active engagement.
- Body and headline font: 'Inter', a sans-serif for clean readability.
- Use simple, outline-style icons for key actions like search and toggle, enhancing the app’s usability without cluttering the interface.
- Maintain a minimalistic design, placing the search bar prominently at the top, followed by the flashcard display, ensuring a clear and focused learning experience.
- Use subtle fade-in animations for the flashcard when generated, providing a smooth and engaging transition.
